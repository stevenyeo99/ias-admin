# PROJECT BRIEF

## Project Name

IAS Automation Console

## Purpose

This project is an internal prototype to demonstrate automation capabilities for insurance operational workflows.

The system will eventually:
- interact with internal web-based systems
- retrieve policy/member data
- display automation activities
- generate CSV output for downstream systems

## Current Phase

Frontend UI mockup prototype only.

## Users

- Internal operations team
- Management demo users
- Automation/AI showcase audience

## Future Scope

Potential future enhancements:
- browser automation
- OCR extraction
- AI-assisted workflows
- CSV integration
- live browser streaming
