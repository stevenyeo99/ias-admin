# React Console UI Skill

## Goal

Build modern enterprise dashboard UI for insurance automation workflows.

## UI Style

- Clean white background
- Blue and green accent colors
- Enterprise operation console feel
- Spacious layout
- Rounded cards
- Soft shadows
- Minimal modern appearance

## Layout

- Left control panel
- Right viewer panel
- Bottom status cards

## Component Rules

- Keep components reusable
- Keep components small
- Use modular structure
- Prefer readable JSX

## Current Phase

Frontend prototype only.

Use:
- mock data
- placeholder viewer
- static logs

Do NOT implement:
- backend integration
- Playwright
- real automation
- real IAS login
